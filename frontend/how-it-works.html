How the System Works

Follow the correct path for registration, reporting, tip submission, and browsing active cases. Each step below mirrors the actual workflow used in the platform.

Register an Account

Create a verified account before you submit cases or track private reports.

  1. Open the Register page and choose your role: public, police, or organization.
  2. Use a valid @gmail.com address and a password with at least 8 characters, letters, and numbers.
  3. If you select police or organization, upload the required identity document and profile photo.
  4. Submit the form and wait for approval if your role requires verification.

Report a Missing Person

Use the official report form to submit a case to the registry.

  1. Log in and open the Report Missing Person page.
  2. Complete the legal check and enter verified case details, location, and subject information.
  3. Upload a clear face photo so the AI service can validate it for matching.
  4. Submit the report to publish it to the registry or send it for approval, depending on your role.

Send a Tip

Provide new information from any case profile without changing the official record yourself.

  1. Open a case profile from the active cases list.
  2. Write what you saw, where you saw it, and when it happened.
  3. Add your contact information if you want investigators to follow up, then submit the tip.

See Only Active Cases

The platform keeps active missing-person cases visible and hides found cases outside police workflows.

  1. Use the homepage search panel to browse public active cases only.
  2. Use the dashboard search and filters to review the active cases linked to your account.
  3. Open each profile to review the last-seen data, classification, and timeline.
  4. Use the refresh control to load the latest active records from the registry.
